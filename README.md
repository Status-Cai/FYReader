# FYReader-master
风月读书，一款开源、无广告的小说阅读软件。

    11个网络小说书源：天籁小说、笔趣阁44、品书网、笔趣阁、
                    全本小说、米趣小说、九桃小说、云中书库、
                    搜小说网、全小说网、奇奇小说
    4个实体书书源：超星图书·实体、作品集·实体、99藏书·实体、100本·实体
    
    发现界面：排行榜、分类、书城
    
    详细功能可查看图片或下载自行体验或自行编译
